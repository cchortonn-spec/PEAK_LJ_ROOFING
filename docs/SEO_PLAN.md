## Internal Linking Plan

Map how pages should link to each other for maximum SEO value.

```
Homepage
├── → Services (hub)
│   ├── → /services/emergency-plumbing
│   ├── → /services/boiler-repair
│   └── → /services/bathroom-installation
├── → Areas (hub)
│   ├── → /areas/manchester
│   ├── → /areas/salford
│   └── → /areas/stockport
├── → About
├── → Contact
└── → Blog
    ├── → /blog/post-1
    └── → /blog/post-2
```

### Linking Rules
- Every service page links to Contact and 2 related services
- Every location page links to all services and Contact
- Blog posts link to relevant service pages
- Homepage links to top 3 services and top 3 locations
