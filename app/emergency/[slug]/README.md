# Emergency Landing Pages (Future)

High-intent emergency service pages will live here.

**URL pattern:** `/emergency/[slug]`

**Example:** `/emergency/emergency-plumber-manchester`

## When building

1. Keep pages short and fast — maximum conversion focus
2. Click-to-call above the fold
3. Minimal sections: hero, trust bar, CTA
4. Unique metadata targeting emergency keywords
5. Register in `app/sitemap.ts`

See `docs/SEO.md` for emergency page strategy.
