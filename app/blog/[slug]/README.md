# Blog / Articles (Future)

Blog posts and articles will live here.

**URL pattern:** `/blog/[slug]`

**Example:** `/blog/how-to-bleed-a-radiator`

## When building

1. Add post metadata to config or a `content/` folder
2. Use `Article` schema markup
3. Include internal links to service and location pages
4. Register in `app/sitemap.ts`

See `docs/SEO.md` for blog strategy.
