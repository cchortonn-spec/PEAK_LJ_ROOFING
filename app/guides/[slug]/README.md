# Guide Pages (Future)

Evergreen guide and resource pages will live here.

**URL pattern:** `/guides/[slug]`

**Example:** `/guides/boiler-maintenance-checklist`

## When building

1. Target long-tail informational keywords
2. Use `Article` or `HowTo` schema where appropriate
3. Link to relevant service pages
4. Register in `app/sitemap.ts`

See `docs/SEO.md` for content strategy.
